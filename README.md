# E-Commerce-Sales-Website-Analytics-Dashboard-Power-BI-
This project analyzes e-commerce transaction and website traffic data to understand business performance.
Using Power BI, an interactive dashboard was created to visualize sales trends, product performance, traffic sources, and refund analysis.
The report contains two dashboard pages for better insight analysis.

Project Objective: The objective of this project is to analyze business data and create an interactive dashboard that helps stakeholders make data-driven decisions.
